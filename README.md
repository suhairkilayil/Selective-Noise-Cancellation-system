# 🎧 Selective Noise Cancellation Using Machine Learning

An intelligent audio filtering system that uses **Machine Learning and Deep Learning** to selectively suppress unwanted background noise while preserving important sounds such as **🚨 sirens, 🚗 car horns, and 🔔 alarms**.

Traditional Active Noise Cancellation (ANC) removes all surrounding sounds without distinction. This project introduces a **context-aware noise cancellation system** that identifies environmental sounds and filters only irrelevant noise while maintaining critical audio cues.

---

## 🚀 Key Features

- 🔊 **Source separation using Demucs**
- 🧠 **CNN-based sound classification**
- 🎼 Audio feature extraction using **MFCC and spectral features**
- 🎯 Selective suppression of non-important background noise
- 🎧 Reconstruction of clean and meaningful audio output

---

## ⚙️ System Workflow

Audio Input → Source Separation → Feature Extraction → Sound Classification → Selective Noise Filtering → Clean Audio Output

---

## 📊 Dataset

The system is trained using the **UrbanSound8K dataset** which contains:

- 🎵 **8732 labeled audio clips**
- 🏙️ **10 classes of urban environmental sounds**
- Real-world sound recordings

Examples include:
- 🚨 Siren  
- 🚗 Car horn  
- 🐶 Dog bark  
- 🔧 Drilling  
- 🚙 Engine idling  
- 🎶 Street music
